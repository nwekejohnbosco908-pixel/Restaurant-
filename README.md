# Restaurant Website
This is a modern, responsive restaurant website designed to showcase menu items, allow reservations, and provide information about the restaurant.
It was built with a focus on usability, accessibility, and responsive design to ensure a great experience across all devices.

## Features
- Responsive Design
- Menu section
- Contact section
- Smooth navigation
- Social media limks

## Tech Stack
- Frontend - HTML5, CSS3, Javascript
- Framework  - Bootstrap
